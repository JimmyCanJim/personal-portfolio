Spam Detection using ML

Description

This personal project is a machine learning-based email classifier that uses the Naive Bayes algorithm to distinguish between spam and ham (legitimate) emails. Currently, it achieves 94% accuracy, effectively filtering unwanted messages while retaining genuine ones. The model is designed to improve with more training data, making it scalable and increasingly reliable over time.
