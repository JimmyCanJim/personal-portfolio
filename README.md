# Personal Portfolio Website (HTML, CSS, JS)

## Overview
This is a fully responsive personal portfolio website built using HTML5, CSS3, and JavaScript. The portfolio serves as a dynamic platform that highlights my growth, skills, and development journey as an aspiring Software Developer and Data Engineer. It features a modern, clean user interface with smooth animations, interactive components, and responsive layouts. This project reflects not only my technical ability, but also my attention to UI/UX design, performance optimization, and structured front-end architecture.

## Website Structure
### The website is structured to present:
- My technical skills and technology stack
- Featured development projects with interactive elements
- Background and professional journey
- Contact Information (Contact Me section)
- Downloadable CV
- Smooth scrolling navigation and section transitions



## Key Features
- Fully responsive design (mobile-first approach)
- Smooth CSS transitions and animations
- Interactive project cards
- Scroll-based reveal effects
- Modern UI styling with glassmorphism elements
- Clean layout with consistent spacing and typography
- Optimized performance and lightweight structure
- Structured and maintainable codebase

## Technologies Used
### Front-End
- HTML5 (Semantic structure)
- CSS3 (Flexbox, transitions, animations, responsive design)
- JavaScript (DOM manipulation, interactivity, event handling)

## Learning Objectives
### This project was built to:
- Strengthen my understanding of responsive web design
- Improve my CSS animation and layout skills
- Practice JavaScript DOM manipulation
- Apply modern UI/UX design principles
- Build a professional online presence

## Deployment
This portfolio website is deployed using __Firebase Hosting__, allowing for fast, secure, and reliable global access.
